# 아메바의 번식

## 문제 설명
신비한 단세포 생물인 **'아메바'**를 관찰하고 있습니다.
실험관에는 **1일 차**에 **갓 태어난 아메바 1마리**가 들어있습니다.

이 아메바의 생애는 다음과 같습니다.

1.  **성장 (Wait)**: 태어난 후 **$D$일 동안은 성장**만 합니다. (번식 불가)
2.  **분열 (Spawn)**: 성장이 끝난 다음 날부터, 살아있는 동안 **매일 1마리씩 새로운 아메바**를 만들어냅니다.
3.  **수명 (Life)**: 태어난 지 **$F$일이 지나면 수명이 다해 사라집니다.**

관찰 기간 $N$일이 지났을 때, 실험관 안에 살아있는 아메바는 총 몇 마리인지 구하세요.
(단, 관찰 기간 $N$은 최대 25일로, 결과값은 아주 크지 않습니다.)

---

## 입력 형식 (Input Format)
* 첫 번째 줄에 관찰 기간 $N$ ($1 \le N \le 25$)이 주어집니다.
* 두 번째 줄에 성장 기간 $D$ ($1 \le D < F$)가 주어집니다.
* 세 번째 줄에 수명 $F$ ($D < F \le N$)가 주어집니다.

## 출력 형식 (Output Format)
* $N$일 차에 살아있는 아메바의 **총 마리 수**를 정수로 출력합니다.

---

## 입출력 예시 (Sample I/O)

### 예시 1
**Input:**
```
6
2
4
```

**Output:**
```
5
```

* **상황 설명**: ($N=6$, 성장 $2$일, 수명 $4$일)
    * **1일**: 아메바(A) 1마리 시작.
    * **2일**: A 성장 중. (총 1마리)
    * **3일**: A 성장 끝. **아기(B)** 탄생. (총 2마리)
    * **4일**: A가 또 **아기(C)** 탄생. B는 성장 중. (총 3마리)
    * **5일**: 
        * A는 수명(4일)이 다해 **사라짐**.
        * B 성장 끝. **아기(D)** 탄생.
        * C 성장 중.
        * (총 3마리: B, C, D)
    * **6일**:
        * B가 **아기(E)** 탄생.
        * C 성장 끝. **아기(F)** 탄생.
        * D 성장 중.
        * (총 5마리: B, C, D, E, F)

### 예시 2
**Input:**
```
4
1
3
```

**Output:**
```
6
```

* **상황 설명**: ($N=4$, 성장 $1$일, 수명 $3$일)
    * **1일**: A (1마리)
    * **2일**: A 성장 끝 $\rightarrow$ B 탄생 (총 2마리)
    * **3일**: A $\rightarrow$ C 탄생, B 성장 끝 $\rightarrow$ D 탄생 (총 4마리)
        * A는 오늘이 3일째. 내일이면 사라짐.
    * **4일**: 
        * A 사라짐. (-1)
        * B $\rightarrow$ E 탄생 (+1)
        * C $\rightarrow$ F 탄생 (+1)
        * D $\rightarrow$ G 탄생 (+1)
        * **계산**: 4(어제) - 1(사망) + 3(탄생) = **6마리**
