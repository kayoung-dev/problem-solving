## description
우주비행사 **리사**는 우주 정거장에 도킹하기 위해 접근 중입니다.<br/>
우주 정거장의 도킹 포트는 $S$라는 문자열로 표현되며, 여기에는 세 종류의 괄호 문양인 $(, ), [, ], {, }$ 가 새겨져 있습니다.<br/>
이 정거장은 일정한 속도로 회전하고 있어서, 리사는 문자열 $S$를 왼쪽으로 $x$만큼 회전시켰을 때 포트가 '올바른 규칙'을 만족하는지 확인해야 합니다.<br/>
문자열을 왼쪽으로 $x$칸 회전시킨다는 것은 첫 번째 문자를 제일 뒤로 보내는 과정을 $x$번 반복하는 것을 의미합니다.<br/>
예를 들어, $S = "[](){}"$ 일 때: $x=0$은 올바름, $x=1$은 올바르지 않음, $x=2$는 올바름입니다.<br/>
리사가 $S$를 $0$칸부터 $n-1$칸($n$은 $S$의 길이)까지 각각 회전시켜 보았을 때, 올바른 괄호 문자열이 되는 $x$의 개수가 총 몇 개인지 계산하는 프로그램을 작성하세요.

## input_description
- 첫 번째 줄에 대괄호, 중괄호, 소괄호로 이루어진 문자열 $S$가 주어집니다.
- $S$의 길이 $n$은 $1 \le n \le 1,000$ 입니다.

## output_description
- $x$번 회전시켰을 때 올바른 괄호 문자열이 되는 경우의 수를 출력합니다.

# samples

### input 1
```
[](){}
```

### output 1
```
3
```


### input 2
```
}}}]
```

### output 2
```
0
```
