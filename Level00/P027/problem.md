## description
가을을 맞아 숲속 다람쥐 '토토'는 겨울잠을 대비해 도토리를 모으고 있습니다.<br/>
토토는 맛있는 도토리를 구별하는 특별한 기준인 **divisor**를 가지고 있습니다.<br/>
토토가 주운 도토리들의 무게가 담긴 배열이 주어졌을 때, 다음 규칙에 따라 도토리를 주머니에 담으려 합니다.
1. 도토리의 무게가 **divisor**로 **나누어 떨어지는 것**만 주머니에 담습니다.
2. 담은 도토리들은 무게가 가벼운 순서인 **오름차순**으로 정렬해야 합니다.
3. 규칙에 맞는 도토리가 하나도 없다면 **-1**을 반환해야 합니다.
토토가 최종적으로 주머니에 담게 될 도토리의 무게 목록을 구하는 프로그램을 작성하세요.

## input_description
- 첫 번째 줄에 도토리의 개수 $N$과 기준 무게 divisor가 공백으로 구분되어 주어집니다. ($1 \le N \le 100$, $1 \le$ divisor $\le 100$)
- 두 번째 줄에 $N$개의 도토리 무게(자연수)가 공백으로 구분되어 주어집니다.

## output_description
- 선택된 도토리의 무게를 오름차순으로 정렬하여 공백으로 구분해 출력합니다.
- 선택된 도토리가 없다면 -1을 출력합니다.

# samples

### input 1
```
4 10
5 20 10 3
```

### output 1
```
10 20
```


### input 2
```
3 7
2 8 5
```

### output 2
```
-1
```
