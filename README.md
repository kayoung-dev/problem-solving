## 📂 1. 표준 디렉토리 구조
새로운 문제가 추가될 때마다 아래의 폴더 구조를 준수합니다.

```text

.
├── Easy/                         # 난이도: 0
│   ├── P01/                      # 문제 번호
│   │   ├── problem.md            # 문제 설명 및 입출력 예시
│   │   ├── solution.py           # 표준 정답 코드
│   │   └── test/                 # 테스트 데이터
│   │       ├── input_01.in
│   │       ├── input_02.in
│   │       ├── output_01.out
│   │       ├── output_02.out
│   │       └── ...               # 입출력 테스트 케이스 20개
│   │
│   ├── P02/
│   │   ├── problem.md
│   │   ├── solution.py
│   │   └── test/
│   │       └── ...
│   │
│   └── ...
│       
│
├── Level01/                       # 1 난이도
│   └── P01/
│       └── ...
│
├── Level02/                       # 2 난이도
│   └── P01/
│       └── ...
│
├── generator/                    # 문제 자동 생성 스크립트
│   ├── easy/                     
│   │   ├── g01.py                # Easy/P01 생성기
│   │   ├── g02.py
│   │ 
│   ├── level01/                       
│   │   ├── g01.py                # Level01/P01 생성기
│   └── ...
│   
├── run.py                        # 전체 generator 실행 스크립트
│
└── README.md                     # 프로젝트 전체 가이드

```

* Level은 숫자가 올라갈 수록 난이도가 높아집니다.
